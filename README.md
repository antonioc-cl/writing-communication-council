# Writing / Communication Council Agent Skill

Expert writing council with 6 advisors for clarity, persuasion, storytelling, and written communication.

## Advisors

- **William Zinsser** - Clarity, non-fiction
- **Stephen King** - Narrative, storytelling
- **Ann Handley** - Digital content, marketing
- **Roy Peter Clark** - Technical craft, tools
- **David Ogilvy** - Persuasive copy
- **George Orwell** - Precision, anti-bullshit

## Installation

### Claude Code
```bash
/plugin add https://github.com/TOPAiiN/writing-communication-council
```

### Claude.ai
Upload the `writing-communication-council` folder or add via skills panel.

## Usage

Activate when you need help with:

- Email writing and editing
- Landing pages and copywriting
- Documentation clarity
- Blog posts and newsletters
- Pitch decks and proposals
- Social media posts
- Difficult communications

### Example Prompts

```
Use writing-communication-council to edit my email
Use writing-communication-council to write landing page copy
Use writing-communication-council to improve my documentation
Use writing-communication-council to make my proposal more persuasive
```
